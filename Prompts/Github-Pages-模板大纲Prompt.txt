## 每页固定结构（所有章节页统一）
1. 顶部固定导航栏
   - 左：网站 Logo（链接首页）
   - 右：所有章节链接，当前章高亮
   - 样式：毛玻璃背景，底部金色细线

2. 面包屑导航
   - 格式：首页 › 知识框架 › 当前章节名

3. 左侧点状导航
   - 垂直排列，对应页内各 section
   - 随滚动自动高亮，hover 显示 section 名称

4. Hero 区域
   - 英文章节副标题（小字母间距）
   - 中文大标题（渐变色）
   - 一句话副标题
   - 2-3 行简介
   - 配合章节主题的图形装饰（SVG / CSS / emoji 组合）

5. 内容 Section
   - 每节含：eyebrow 小标 / 大标题 / 引导段 / 正文内容
   - 各节有唯一 id，供左侧导航锚定
   - 节与节之间用细线分隔

6. Insight 提示块
   - 金色左边框
   - 用于：记忆口诀 / 重点强调 / 易错提示

7. 底部章节翻页
   - 格式：← 上一章标题 | · 章节目录 · | 下一章标题 →

8. Footer
   - 左：网站名
   - 中：当前章节
   - 右：首页 / 目录 / 下一章

9. 返回顶部按钮（右下角悬浮）

10. 滚动渐入动画
    - 使用 IntersectionObserver
    - 卡片按顺序错位延迟出现

## 内容组件（按需选用）

### 信息卡片类
- 标准卡片：图标 + 标题 + 正文 + 标签组
- 对比双栏卡片：左右两列，适合对比两个概念
- 大字符卡片：左侧大字（汉字/符号），右侧信息区

### 表格类
- 全宽关系对应表：多列横向对照（如五脏对应表）
- 分组双栏表：阴/阳、正/反等两组对比

### 流程/序列类
- 横向流程条：多步骤等宽格，适合顺序/循环
- 带箭头流程图：步骤间用箭头连接

### 网格展示类
- 2列网格
- 3列网格
- 4列网格
- 6列等宽横条（适合12项以内的并列内容）

### 层级/嵌套类
- 折叠块：标题区 + 展开内容区（适合大块详细说明）
- 编号原则块：序号徽标 + 标题 + 说明，背景大字水印
